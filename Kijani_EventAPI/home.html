üöÄ Quick Start

1. Register a new user

POST /api/v1/auth/register/ { "username": "johndoe", "email": "john@example.com", "password": "secure_password", "password_confirm": "secure_password", "role": "organizer" }

2. Login to get tokens

POST /api/v1/auth/login/ { "username": "johndoe", "password": "secure_password" }

3. Create an event

POST /api/v1/events/ Authorization: Bearer your_access_token { "title": "Tech Workshop", "description": "Learn to code", "event_date": "2025-12-01T14:00:00Z", "location": "Nairobi", "capacity": 50 }
